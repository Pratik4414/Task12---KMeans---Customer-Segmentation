# KMeans Customer Segmentation

## Objective
Perform customer segmentation using KMeans clustering for business insights.

## Dataset
Mall Customer Segmentation Dataset (Kaggle)

## Steps Performed
1. Loaded and inspected dataset
2. Dropped CustomerID
3. Applied StandardScaler
4. Used Elbow Method to find optimal K
5. Applied KMeans clustering
6. Visualized clusters
7. Interpreted customer segments

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Output
- Elbow plot
- Cluster visualization
- Segmented customer CSV
